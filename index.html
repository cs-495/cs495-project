Project Description

DMAGIC (Drawing, Music, Art, Game, and Idea Creator) (Codename: Kitty Paint) is an android application aimed for users who wish to have the ability to create simple creative projects together on the go. For this project we are aiming to create an interface for image editing—with the possibility of adding drag and drop music composition, and animation editing—that allows users to connect and edit the same project on seperate phones/platforms simultaneously online. Users should be able to save their visual, audio, or multimedia creations in formats natively useable by android, and also in raw formats that can be shared with others to edit or remix at a later time.
